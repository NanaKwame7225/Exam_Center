CSV Format Guide

Your CSV must have these columns in order:
course, exam_type, question, type, option_a, option_b, option_c, option_d, correct_answer, marks

type values: mcq, tf, or short
correct_answer: for MCQ use a, b, c or d · for T/F use true or false · for short, type the model answer
For T/F and short answer questions, leave option columns empty.
